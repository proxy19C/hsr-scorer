# 崩壊：スターレール キャラクター評価ツール

**言語選択 / Language / 语言选择:** [中文](README.md) | [English](README.en.md) | [日本語](#)

崩壊：スターレールのファンがゲームキャラクターを互いに比較できる小さなWebアプリケーションです。

## 開発環境のセットアップ方法

Node Version Manager (nvm) の使用を強く推奨します。そうでなければ `.nvmrc` を確認し、お使いのNodeのバージョンが一致していることを確認してください。

1. 依存関係をインストール `npm install`
2. 開発サーバーを起動 `npm run dev`

貢献したい方へのメッセージ！

1. ESLintルールに従うようにしてください。無視しないでください。
2. 大きなコミットは避け、変更を複数のコミットまたは複数のプルリクエストに分割してください。
