# 崩坏：星穹铁道角色评分器

**语言选择 / Language / 言語:** [中文](#) | [English](README.en.md) | [日本語](README.ja.md)

一个小型网页应用，让崩坏：星穹铁道玩家可以互相比较游戏角色。

## 如何设置开发环境

我强烈推荐使用 Node 版本管理器 (nvm)，否则请检查 `.nvmrc` 文件并确保你的 Node 版本与之匹配。

1. 安装依赖 `npm install`
2. 启动开发服务器 `npm run dev`

1. 请确保遵循 ESLint 规则，尽量不要忽略它们。
